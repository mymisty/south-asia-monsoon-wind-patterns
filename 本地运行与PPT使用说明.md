# 南亚季风互动模型：本地运行与 PPT 使用

## 本地运行

直接双击打开：

```text
monsoon-model-local.html
```

这个文件是独立网页，不需要微信开发者工具、不需要安装依赖、不需要联网。

网页会自动记住上次的课堂状态，包括所在页面、冬夏季节、图层开关和测验进度。若需要重新开始，进入“互动模型”页点击“重置课堂状态”。

底图已换成 NASA Blue Marble Next Generation 真实卫星地形底图，1月对应冬季图，7月对应夏季图。来源记录见：

```text
assets/nasa/SOURCE.md
```

## 放进 PPT 的方式

推荐两种：

1. **课堂互动演示**
   - 在 PPT 中给文字或图片添加超链接，链接到 `monsoon-model-local.html`。
   - 上课时点击链接，浏览器打开互动模型。

2. **静态图插入**
   - 打开 `monsoon-model-local.html`。
   - 切到“互动模型”页，调整冬季/夏季和图层开关。
   - 点击“导出当前模型图”，得到 PNG 图片。
   - 把 PNG 插入 PPT。

说明：PowerPoint 原生“插入对象”通常不能稳定运行本地交互式 HTML。若要在幻灯片内部直接交互，一般需要 Web Viewer / LiveWeb 类插件，或者把网页放到本地/在线服务器后再嵌入。
